# Simple Clean Architecture iOS App:

## Production:

- Task, `async`/`await`
- Combine
- SwiftUI
- MVVM pattern
- GRDB persistence library
- Alamofire
- Kingfisher
- Lottie Animations

## Testing:
- Cuckoo - Swift Mock Library
- Nimble - Swift Matcher Framework
